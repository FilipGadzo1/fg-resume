# fg-resume

Personal portfolio and resume site for Filip Gadžo — built with React, TypeScript, and Tailwind CSS.

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion

## Projects showcased

- **Budget Dashboard** — full-stack finance app with Vue 3, Supabase, real-time collaboration, and Google OAuth
- **Skye** — weather app with animated backgrounds, 7-day forecast, air quality, and AI-generated summaries

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Deployed on Vercel. Push to `master` to trigger a new deployment.
